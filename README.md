# password-generator-v2
Single-file HTML password generator. It uses JavaScript for all logic and CSS for basic styling.

# Required Features:
1. An input range slider for password length (from 8 to 128 characters). 2.
2. A set of checkboxes to include/exclude the following character types:
      Uppercase letters (A-Z)
      Lowercase letters (a-z)
      Numbers (0-9)
      Symbols (!@#$%^&*)
3. A 'Generate Password' button and a read-only text field to display the result.
4. A small 'Copy to Clipboard' button next to the result field.
5. The code should be well-commented, responsive, and follow modern JavaScript (ES6) conventions.
